# 五教科学習・1対1対戦RPG　統合仕様書

## 1．ゲーム概要

| 項目    | 内容                               |
| ----- | -------------------------------- |
| ジャンル  | 五教科の選択問題と、プレイヤー同士の対戦を組み合わせた学習RPG |
| 対戦形式  | 中学3年生同士による1対1のターン制対戦             |
| 基本形式  | プレイヤーは移動せず、交互に技を出し合う             |
| 対象学年  | 中学3年生                            |
| 出題範囲  | 中学1年生から中学3年生の1学期まで               |
| 問題レベル | 中学1・2年生の内容も、中学3年生向けの難しさで出題する     |
| 教科    | 国語・数学・英語・理科・社会の五教科               |
| 大会の教科 | 大会の主催者が、大会開始前に一つの教科を選ぶ           |
| 勝利条件  | 相手のHPを0にしたプレイヤーの勝利               |
| 初期HP  | 120                              |
| MP    | 使用しない                            |
| 選択肢   | すべて4択                            |
| 解答時間  | 技1回につき20秒                        |
| 技の数   | 4個                               |

---

## 2．対戦の基本的な流れ

| 順番 | 内容                            |
| -: | ----------------------------- |
|  1 | 行動するプレイヤーが、使用する技を選ぶ           |
|  2 | 選んだ技を確認し、決定する                 |
|  3 | 技に対応する難易度の4択問題が始まる            |
|  4 | 20秒間で、できるだけ多くの問題に解答する         |
|  5 | 正解数と誤答数から有効得点を計算する            |
|  6 | 有効得点によって技のダメージ・回復量・防御効果が強化される |
|  7 | 技の発動演出を行い、効果を反映する             |
|  8 | 相手プレイヤーのターンに移る                |
|  9 | どちらかのHPが0になるまで繰り返す            |

技の選択時間は10秒とする。

---

## 3．得点の計算

### 有効得点

**有効得点 ＝ 正解数 − 誤答数 × 0.5**

有効得点が0未満になった場合は、0として扱う。

| 解答結果            | 得点への影響 |
| --------------- | -----: |
| 正解              |    ＋1点 |
| 誤答              |  −0.5点 |
| 無回答             |   変化なし |
| 時間切れ時に表示されていた問題 |    無回答 |
| 1問も解答しなかった場合    |  有効得点0 |

### 無回答の処理

* 無回答による追加ペナルティは発生しない。
* 1問も解答しなかった場合、技の追加強化は発生しない。
* 技の基本効果は、有効得点が0でも発動する。
* 1問以上解答したあと、次の問題を解いている途中で時間切れになった場合、その途中の問題は無回答として扱う。

---

## 4．使用できる技

技名は、特定の教科・属性・性別などに偏らず、効果が分かりやすい名前で統一する。

| 技名   | 種類   | 効果                    | 効果の計算         | 待ち時間    |
| ---- | ---- | --------------------- | ------------- | ------- |
| アタック | 通常攻撃 | 相手に安定したダメージを与える       | 8＋有効得点×4ダメージ  | なし      |
| リカバー | 回復   | 自分のHPを回復する            | 6＋有効得点×3回復    | 使用後1ターン |
| ガード  | 防御   | 次に受ける攻撃のダメージを減らす      | 15＋有効得点×7％軽減  | 使用後1ターン |
| バースト | つよ技  | 難しい問題に挑戦し、大きなダメージを与える | 16＋有効得点×8ダメージ | 使用後1ターン |

### 技の追加ルール

* ガードの軽減率は最大60％とする。
* ガードの効果は、次に攻撃ダメージを受けるまで継続する。
* リカバーを使用しても、HPは最大HPの120を超えない。
* 「使用後1ターン」とは、使用した次の自分のターンでは選べないことを意味する。
* 待ち時間が終了すると、再び使用できる。
* ダメージ、回復量、軽減率に小数が出た場合は四捨五入する。

---

## 5．技と問題難易度

| 技    | 問題難易度 |
| ---- | ----- |
| アタック | 通常    |
| リカバー | 通常    |
| ガード  | 通常    |
| バースト | 難しい   |

バーストは「つよ技」と「大技」を統合した技である。

難しい問題では、通常問題よりも次の要素を増やす。

* 複数の知識を組み合わせる
* 考える手順が多い
* 資料・表・グラフなどを読み取る
* よく似た選択肢から正確に判断する
* 基礎知識を応用して答える

出題範囲外の内容は使用しない。

---

## 6．五教科の問題方針

問題例や実際の問題文は仕様書には記載せず、どのような問題を出題するかだけを定める。

| 教科 | 出題する問題の種類                                  |
| -- | ------------------------------------------ |
| 国語 | 漢字の読み・意味、語句、文法、慣用句、ことわざ、短文読解、古文の基礎         |
| 数学 | 計算、方程式、比例・反比例、関数、図形、確率、資料の活用、式の展開・因数分解、平方根 |
| 英語 | 英単語、熟語、文法、空欄補充、語順、英文の意味、短い会話文、短文読解         |
| 理科 | 生物・化学・物理・地学の知識、実験、結果の予想、表・グラフ・図の読み取り、簡単な計算 |
| 社会 | 地理・歴史・公民、出来事の順番、地図、統計、年表、資料の読み取り           |

### 問題作成の共通ルール

| 項目         | 方針                           |
| ---------- | ---------------------------- |
| 問題形式       | 短時間で解答できる4択問題                |
| 正解数        | 選択肢のうち正解は一つ                  |
| 誤答選択肢      | 実際に間違えやすい考え方を反映する            |
| 問題文        | 必要以上に長くしない                   |
| 中学1・2年生の内容 | 単純な暗記だけでなく、中学3年生向けの判断・応用を含める |
| 通常問題       | 基礎知識と標準的な思考問題を中心にする          |
| 難しい問題      | 複数知識の組み合わせ、応用、資料読解を中心にする     |
| 公平性        | 両プレイヤーで問題数や難易度に差が出ないようにする    |
| 解説         | 対戦終了後に、正解と短い解説を確認できるようにする    |

# 7．ダメージ演出

## 基本方針

実際に受けたダメージが大きいほど、次の演出を強くする。

* ダメージ効果音
* キャラクターの赤色変化
* キャラクターの揺れ
* ダメージ数字
* HPゲージの減少演出
* 特大ダメージ時の一瞬停止

ダメージ段階は、ガードなどによる軽減を計算したあとの最終ダメージで判定する。

---

## 8．ダメージの4段階

最大HP120を基準として、ダメージ演出を4段階に分ける。

| 段階     | 最終ダメージ | 最大HPに対する割合 | 演出    |
| ------ | -----: | ---------: | ----- |
| 小ダメージ  |   1～11 |      10％未満 | 控えめ   |
| 中ダメージ  |  12～23 | 10％以上20％未満 | 標準    |
| 大ダメージ  |  24～41 | 20％以上35％未満 | 強い    |
| 特大ダメージ |   42以上 |      35％以上 | とても強い |

同じ段階の中でも、ダメージが大きくなるほど音量・赤色・揺れ幅を少しずつ強くする。

---

## 9．ダメージ演出の詳細

| 演出項目      | 仕様                        |
| --------- | ------------------------- |
| ダメージ効果音   | ダメージが大きいほど音量を大きくする        |
| 大ダメージの音   | 大ダメージ以上では、低い音や重い衝撃音を追加する  |
| 赤色変化      | ダメージを受けたキャラクターを0.25秒間赤くする |
| 赤色の濃さ     | ダメージが大きいほど赤色を濃くする         |
| キャラクターの揺れ | ダメージが大きいほど揺れ幅を大きくする       |
| 揺れ時間      | 最大0.4秒                    |
| 揺れる対象     | ダメージを受けたキャラクターだけ          |
| ダメージ数字    | キャラクターの上に実際のダメージ量を表示する    |
| 大ダメージの数字  | 数字を一度大きくしてから消す            |
| 特大ダメージ    | 数字を大きく拡大し、一瞬停止と組み合わせる     |
| 一瞬停止      | 特大ダメージ時のみ約0.1秒停止する        |
| HPゲージ     | 現在HPのゲージを先に減らす            |
| HPゲージの残像  | 減った部分を白色または黄色で一瞬残す        |
| 残像の動き     | 少し遅れて現在HPの位置まで縮小する        |
| 演出設定      | 開発時間を優先し、演出を弱める設定は実装しない   |

### 段階別の演出

| 演出   | 小      | 中      | 大      | 特大       |
| ---- | ------ | ------ | ------ | -------- |
| 効果音  | 小さな命中音 | 通常の衝撃音 | 大きく重い音 | 最も大きく重い音 |
| 赤色   | 薄い     | 標準     | 濃い     | 最も濃い     |
| 赤色時間 | 0.25秒  | 0.25秒  | 0.25秒  | 0.25秒    |
| 揺れ幅  | ごく小さい  | 小さい    | 大きい    | 最も大きい    |
| 揺れ時間 | 短い     | やや短い   | 最大0.4秒 | 最大0.4秒   |
| 数字   | 通常表示   | 通常表示   | 一度拡大   | 大きく拡大    |
| 一瞬停止 | なし     | なし     | なし     | 約0.1秒    |
| HP残像 | あり     | あり     | あり     | あり       |

### 演出上の注意

* 問題文や選択肢は揺らさない。
* 画面全体は揺らさない。
* 画面全体を赤く点滅させない。
* キャラクターが画面外にはみ出すほど揺らさない。
* 効果音が突然極端に大きくならないようにする。
* 特大ダメージの一瞬停止は、操作や得点計算には影響させない。

## 13．問題解答画面

| 表示内容  | 仕様                |
| ----- | ----------------- |
| 使用中の技 | 画面上部に技名とアイコンを表示する |
| 問題文   | 画面中央に大きく表示する      |
| 選択肢   | 押しやすい大きなボタンにする    |
| 残り時間  | 常に確認できる位置に表示する    |
| 正解数   | リアルタイム表示          |
| 誤答数   | リアルタイム表示          |
| 有効得点  | リアルタイム表示          |
| 技ボタン  | 問題解答中は非表示にする      |
| 次の問題  | 解答後、自動的に切り替える     |

### 正誤表示

* 正解した場合、選んだ選択肢を一瞬明るくする。
* 誤答した場合、選んだ選択肢を一瞬暗くする。
* 誤答時は正しい選択肢も短く表示する。
* 表示後はすぐに次の問題へ移る。
* 正誤表示によって対戦のテンポを大きく止めない。

## 14．技発動画面

20秒の解答時間が終了したら、次の情報を短く表示する。

| 表示内容  | 例       |
| ----- | ------- |
| 正解数   | 正解：6問   |
| 誤答数   | 誤答：2問   |
| 有効得点  | 有効得点：5点 |
| 使用した技 | アタック    |
| 計算    | 8＋5×4   |
| 最終効果  | 28ダメージ  |

計算結果を表示したあと、キャラクターが技を発動し、ダメージ・回復・防御の演出へ移る。

# 17．全体の重要方針

* 学習問題の正解数が、技の強さに直接つながる。
* 誤答には減点があるが、無回答には追加ペナルティを与えない。
* 問題を解くことと技を選ぶことが複雑になりすぎないよう、MPは使用しない。
* 技は4個に絞り、それぞれの役割を分かりやすくする。
* つよ技のバーストだけ、難しい問題を使用する。
* ダメージが大きいほど、音・赤色・揺れ・数字・HPゲージの演出を強くする。
* 問題文や選択肢は、戦闘演出の影響で読みにくくならないようにする。
* 色だけに頼らず、技名・アイコン・数値でも情報を伝える。
* 対戦終了後は正解と解説を確認でき、学習につなげられるようにする。
